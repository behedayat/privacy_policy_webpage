1. Introduction

Livo App is a modern real estate platform designed to simplify the process of searching, viewing, and selecting properties. The application provides users with an intuitive and efficient experience to explore houses, apartments, and land based on their preferences and requirements. Our goal is to connect buyers and sellers through a secure, fast, and user-friendly environment while ensuring transparency and reliability in all property listings. This platform includes advanced tools such as smart search filters, interactive maps, favorites management, and real-time communication features. Livo App is committed to protecting user data, ensuring account security, and continuously improving the overall experience for all users.

  • User Consent & Agreement: By using Livo App, users agree to our terms, privacy policy, and data usage rules designed to ensure a safe and transparent platform.
  • User Rights: Users have full rights to search properties, save favorites, contact sellers, and manage their personal information securely within the application.
  • User Responsibilities: Users must provide accurate information, avoid fraudulent activity, and use the platform in a respectful and legal manner.
  • Data Protection & Security: We apply modern security measures to protect user data, prevent unauthorized access, and ensure safe communication between users and sellers.
  • Platform Purpose: Livo App is designed only for real estate purposes, including buying, selling, and renting properties such as houses, apartments, and land.
2. Information Collection

Livo App collects certain types of information to provide a better real estate experience, improve platform functionality, and ensure secure communication between users. This information is collected only for legitimate purposes such as property searching, account management, and service improvement.

  • Personal Information: We may collect user details such as name, phone number, email address, and profile information during registration and account setup.
  • Property & Activity Data: Information related to user interactions such as saved properties, searches, viewed listings, and favorite items may be collected to improve recommendations.
  • Location Data: With user permission, location information may be used to show nearby properties and enable map-based property search features.
3. Personal Information

Livo App collects and processes personal information only to provide essential services and improve the user experience within the platform. All personal data is handled with strict confidentiality and is not shared with unauthorized third parties.

  • Account Details: We collect basic information such as full name, phone number, email address, and password during registration to create and manage user accounts.
  • Profile Information: Users may optionally provide additional details such as profile picture and preferences to enhance their experience on the platform.
  • Communication Data: Information shared through in-app chat, messages, or contact forms may be stored to facilitate communication between users and property owners.
  • Usage Purpose: Personal information is used for authentication, security verification, property recommendations, and improving overall app performance.
4. Property Listings

Livo App provides a platform for listing, browsing, and discovering real estate properties including houses, apartments, and land. All property listings are displayed based on user preferences, location, and search filters to improve the browsing experience.

  • Listing Information: Property details such as price, location, size, images, and features are provided by property owners or verified users.
  • Content Accuracy: While we aim to ensure accurate listings, users are responsible for verifying property details before making any decisions.
  • Search & Filtering: Users can search properties using advanced filters such as price range, number of rooms, property type, and location.
  • Map Display: Properties may be displayed on an interactive map to help users easily locate and explore nearby listings.
  • Platform Role: Livo App acts as an intermediary platform and does not directly own or guarantee any listed properties.
5. Location Services

Livo App uses location services to enhance the real estate search experience by showing relevant properties based on the user's current or selected location. Location data is only accessed with user permission and can be disabled at any time from device settings.

  • Nearby Properties: Location services help users find properties available near their current area for faster and more relevant search results.
  • Map Functionality: The app uses location data to display properties on an interactive map and allow users to explore specific regions.
  • User Control: Users have full control over location permissions and can enable or disable access at any time.
  • Data Usage: Location information is used only to improve search results and is not shared with unauthorized third parties.
  • Privacy Protection: Livo App ensures that location data is handled securely and is not stored longer than necessary for functionality purposes.
